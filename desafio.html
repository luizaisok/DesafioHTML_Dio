O que é o HTML

O HTML é uma linguagem de computador essencial para a maioria das páginas da internet e aplicativos online. Hipertexto refere-se a textos que fazem referência a outros textos, enquanto uma linguagem de marcação utiliza uma série de tags para definir o estilo e a estrutura de um documento para os servidores da web.O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.Aqui estão alguns dos usos mais comuns do HTML:

  • Desenvolvimento web: Desenvolvedores utilizam HTML para definir como os navegadores devem exibir elementos das páginas, como textos, hyperlinks e arquivos de mídia.
  • Navegação na internet: HTML facilita a navegação e a criação de links entre páginas e sites relacionados, permitindo uma estrutura de hiperlinks eficiente.
  • Documentação: HTML permite organizar e formatar documentos de maneira semelhante ao que é feito no Microsoft Word.
Além disso, o HTML é agora um padrão oficial da internet. O World Wide Web Consortium (W3C) é responsável por manter e desenvolver as especificações do HTML, além de fornecer atualizações regulares.

História

O HTML foi criado por Tim Berners-Lee, um físico que trabalhava no CERN, na Suíça. Ele concebeu a ideia de um sistema de hipertexto para a internet. Hipertexto é um tipo de texto que inclui referências (links) para outros textos, permitindo o acesso imediato a esses conteúdos.
Em 1991, Berners-Lee lançou a primeira versão do HTML, que tinha 18 tags. Desde então, cada nova versão do HTML trouxe novas tags e atributos (modificadores de tags). Com a rápida ascensão e popularidade da linguagem, o HTML tornou-se um padrão oficial da web.
A maior atualização da linguagem foi a introdução do HTML5 em 2014, que adicionou várias novas tags semânticas, como article, header e footer, que ajudam a definir o significado do conteúdo.

Como funciona

Um site típico é composto por várias páginas HTML distintas. Por exemplo, pode incluir uma página inicial, uma página "sobre" e uma página de contato, cada uma representada por um arquivo HTML separado. Os documentos HTML são arquivos que têm a extensão .html ou .htm.
Quando um navegador acessa um arquivo HTML, ele interpreta e exibe o conteúdo para os usuários da internet. Cada página HTML contém uma série de elementos, que são formados por um conjunto de tags e atributos. Esses elementos são os blocos fundamentais de construção de uma página da web. As tags definem onde um elemento começa e termina, enquanto os atributos descrevem as características desse elemento.
As três principais partes de um elemento HTML são:

  • Tag de abertura: Indica o início do efeito de um elemento. É cercada por colchetes angulares. Por exemplo, a tag de abertura p é usada para criar um parágrafo.
  • Conteúdo: É a parte visível para os usuários.
  • Tag de fechamento: Semelhante à tag de abertura, mas com uma barra antes do nome do elemento. Por exemplo, /p é usado para fechar um parágrafo.

O que são as tags?

As tags HTML são elementos usados para marcar e formatar o conteúdo em documentos HTML, tanto textual quanto visual. Cada tag possui um nome e uma sintaxe específica que define como ela deve ser aberta e fechada.

Por exemplo:

A tag de parágrafo é representada por p para iniciar e /p para finalizar. Para negrito, você pode usar strong ou b no início e /strong ou /b para encerrar. Para fechar uma tag, utiliza-se uma barra (/) antes do nome da tag na tag de fechamento. No entanto, algumas tags são auto-fechantes e não requerem uma tag de fechamento. Essas tags não precisam da barra, como por exemplo img para imagens e br para quebras de linha.
